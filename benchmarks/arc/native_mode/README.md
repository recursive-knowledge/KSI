# ARC Native Mode

This mode follows the prompt-and-submission structure used by:

- `benchmarks/arc/benchmarking/`

It is intended for text-first agents and baseline runs that do not need the browser workspace.

The key design is:

- one canonical ARC pair payload is shared by both native mode and UI mode
- native mode derives a text prompt from that payload
- UI mode renders that same payload directly in the browser workspace

Prompts are generated by the upstream benchmarking repo's prompt builder
(`arc_agi_benchmarking.prompts.prompt_manager.convert_task_pairs_to_prompt`)
via `benchmarks/scripts/arc_prep/prepare_arc_native_prompts.py` — there are no static
prompt template files in this directory.

## Canonical payload

The canonical ARC pair payload comes from:

- `benchmarks/scripts/arc_prep/prepare_arc_workspace_payloads.py`

Example shape:

```json
{
  "benchmark": "arc1",
  "task_id": "845d6e51",
  "pair_index": 0,
  "train": [...],
  "test_input": [[...]]
}
```

This is the shared benchmark-safe task artifact.

## What the native agent receives

Native mode derives one prompt file per `task_id + pair_index`:

```json
{
  "benchmark": "arc1",
  "task_id": "845d6e51",
  "pair_index": 0,
  "prompt": "You are participating in a puzzle solving competition..."
}
```

The important field is `prompt`, which is generated using the ARC benchmarking repo's prompt builder from the canonical payload.

## Preparing canonical payloads

```bash
uv run python benchmarks/scripts/arc_prep/prepare_arc_workspace_payloads.py \
  --task-map benchmarks/arc1/task_maps/arc1_train_50_seed0.json
```

Outputs go under the shared payload directory:

- `benchmarks/arc/workspace_payloads/<benchmark>/<task_map_stem>/`

## Rendering native prompts

```bash
uv run python benchmarks/scripts/arc_prep/prepare_arc_native_prompts.py \
  --payload-manifest benchmarks/arc/workspace_payloads/arc1/arc1_train_50_seed0/manifest.json
```

Outputs go under:

- `benchmarks/arc/native_prompts/<benchmark>/<task_map_stem>/`

## What the agent should return

The exported prediction format is the same as UI mode:

```json
{
  "benchmark": "arc1",
  "task_id": "845d6e51",
  "pair_index": 0,
  "attempt_index": 1,
  "prediction": [[1, 0], [0, 1]]
}
```

## Converting and scoring

Use the shared converter:

```bash
uv run python benchmarks/scripts/arc_prep/convert_arc_workspace_predictions.py \
  --predictions-dir /path/to/predictions \
  --output-submission-dir /path/to/submissions \
  --manifest benchmarks/arc/workspace_payloads/arc1/arc1_train_50_seed0/manifest.json
```

Then score with:

```bash
PYTHONPATH=benchmarks/arc/benchmarking/src \
uv run python benchmarks/arc/benchmarking/src/arc_agi_benchmarking/scoring/scoring.py \
  --task_dir benchmarks/arc1/source/data/training \
  --submission_dir /path/to/submissions \
  --results_dir /path/to/results
```
